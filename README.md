# 🌱 Projeto Aprimore

O Aprimore é uma plataforma web desenvolvida com o objetivo de promover a conscientização sobre saúde mental por meio da divulgação de palestras, rodas de conversa e outras atividades educativas.

A plataforma busca centralizar informações sobre eventos relacionados ao tema, oferecendo uma experiência acessível, intuitiva e responsiva para os usuários.

## 🎯 Objetivo

O projeto Aprimore visa fomentar momentos voltados à discussão de **saúde mental**, **qualidade de vida** e **direcionamento nos estudos** de estudantes universitários, contribuindo para uma vivência acadêmica mais leve e equilibrada.
